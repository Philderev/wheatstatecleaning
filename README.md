# wheatstatecleaning

Marketing website for **Wheat State Cleaning** — Lawrence, KS's top-rated residential house-cleaning service.

A static, single-page site (plus Privacy Policy and Terms pages) built with plain HTML, CSS, and vanilla JavaScript — no framework or build step.

## Structure

| Path | Description |
|------|-------------|
| `index.html` | The main landing page |
| `styles.css` | All styling |
| `script.js` | Tabs, mobile menu, review/marquee, and other interactions |
| `services/` | Service-specific landing pages |
| `service-area/` | City/service-area landing pages |
| `pricing.html` | Pricing page |
| `careers.html` | Careers page |
| `thank-you.html` | Form thank-you page |
| `legal/` | Privacy Policy and Terms pages |
| `icons/` | Logo and brand icons |
| `images/` | Photography |
| `fonts/` | Local SF Pro Display font files |

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```
